# AlgoMotion

Code. Run. Visualize.

## 🚀 Vision
AlgoMotion is a desktop application that helps students learn Data Structures and Algorithms by visualizing C++ code execution with animations.

## 🎯 MVP Features
- C++ Code Editor
- Run Button
- Terminal Output
- Array Visualization
- Step-by-Step Animation

## 🛠 Tech Stack
- React
- Electron
- C++
- TypeScript
